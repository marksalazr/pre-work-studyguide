HTML

  • Elemet is a type of content on a webpage. Common elements are as follows
    1. Text elements
    2. image elements
    3. header elemets
    4. list elements
    5. table elements
  • Elements are also used to divide the content on the page and decribes content to other deveolpers
  • The Head element has infotmation about the page. Most common nested tags inside this are things like Meta chraset and title.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users. Metadata is typically not seen by users, but it contains important information about accessibility, search engines, and performance.
  • Body element represents visible content shown to the user. has tags like P (with Brackets)
  • HMTML stand for hyper text markup language and one of the 3 main languaegs used in web development.
  • The !Doctype tag is not case senstive and sits at the very top. This tells the browser what document type is expectedto be opened.
  • IMPORTANT HTML attributes are always defined in the beginning element, never the clsoing

JavaScript

  • Variables are containers that store values. You declare a variable with "let" keyword.

  • Number: Just a number. Does not require to be enclosed in marks.
  • string: A sequence of text enclosed in single quote marks.
  • Array: A structure that allows you to store mulipe values in a refernce.
  • Conditionals are code structures used to test if an expression returns true or not. A very common form of conditionals is the if...else statement
  • Control flow is the order in which a computer executes code in a script
